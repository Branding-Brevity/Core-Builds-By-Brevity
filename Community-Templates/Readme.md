# 🤝 Community Builds

Welcome to the Sandbox. This folder contains experimental, alternative, and user-submitted AIOStreams formatters that fall outside the official **Core Nexus** design language.

### ⚠️ Disclaimer
These templates are provided "as-is." While they offer awesome alternative aesthetics (like the Auburn Tiger edition), they are not subjected to the same rigorous character-limit and crash-testing as the official Flagship builds. Expect occasional formatting quirks.

### 🛠️ How to Contribute
Got a killer layout you want to share? 
1. Fork this repository.
2. Add your `.json` file to this folder.
3. Open a Pull Request! Please include a screenshot of what your formatter looks like in the PR description.
