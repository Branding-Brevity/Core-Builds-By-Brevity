# 🎨 Visual Customization: Swapping Formatters

The **Core Builds** repository includes custom visual layouts (Formatters) designed to make your stream list easier to read at a glance. You can swap these styles instantly using the AIOStreams Formatter Import tool.

---

## 🛠️ Step 1: Download Your Style
Navigate to the [`/Formatters`](../Formatters) folder in this repository and download the `.json` file for your preferred look:

* **Core Zenith Diamond:** Features hardware-specific icons (e.g., 🟣 4K UHD, 💎 Remux, 🎶 JBL Spatial) for instant technical confirmation.
* **Auburn Tiger Edition:** Uses a high-contrast Orange & Blue alternating color scheme with geometric spacers for maximum readability.

---

## ⚙️ Step 2: Open the Formatter Import Menu
1. Open your **AIOStreams Dashboard**.
2. Go to the **Formatter** section.
3. In the bottom right corner, tap the **Import/Export icon** (the box with the arrow pointing inward).

---

## 📄 Step 3: Import the File
1. When the pop-up appears, select **Import from File**.
2. Select the `formatter.json` file you downloaded from this repository.
3. The Name and Description templates will automatically populate with the optimized Core Build code.

---

## 💾 Step 4: Save & Refresh
1. Click **Save** at the bottom of your AIOStreams dashboard.
2. Refresh **WuPlay** or **Stremio**. Your new visual style will be applied instantly to the stream list!

---

> 💡 **Developer Tip:** If you've made your own custom tweaks and want to share them, tap the **Export** icon next to the import button to save your current setup as a JSON file!
