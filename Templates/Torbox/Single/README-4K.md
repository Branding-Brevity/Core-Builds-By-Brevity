## 🎯 Purpose

The unleashed edition for home theater enthusiasts running TorBox. Hunts for the highest-fidelity 4K content — Dolby Vision Remuxes, HDR10+, lossless TrueHD/Atmos audio — with no bitrate caps and AV1 enabled.

## ⚙️ What It Does

- **Resolution priority: 2160p** — falls back to 1080p, blocks 1440p and Unknown
- **BluRay REMUX preferred** — quality first, no compromises
- **Visual tags preferred:** DV → HDR+DV → HDR10+ → HDR10 → HDR → HLG → SDR
- **AV1 encode enabled** — modern efficiency for 4K streams
- **Audio uncapped:** TrueHD, Atmos, DTS:X, DTS-HD MA, FLAC all welcome
- **7.1 channels enabled** — full surround support
- **Uncapped bitrate up to 150Mbps** — for true 4K Remux quality
- **Size up to 150GB movies / 80GB series** — Remuxes accommodated
- **Upscaled 4K blocked** — only native 4K, no upscales pretending to be UHD
- **Bad 4K Anime filtered** — anime-specific quality enforcement
- **Cached streams only · YouTube triple-block · CAM hard-kill**
